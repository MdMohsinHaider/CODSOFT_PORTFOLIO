Skills

HTML (Hypertext Markup Language):
Markup language used for creating the structure of web pages

Consists of elements and tags to define different parts of a webpage, such as headings, paragraphs, images, links, etc.

Provides the basic structure that is enhanced and styled using CSS.
JavaScript:
A versatile scripting language that enables interactive and dynamic content on websites.

Runs on the client-side, making it possible to manipulate the Document Object Model (DOM) and respond to user actions.

Widely used for creating interactive features, form validation, and asynchronous communication with servers (AJAX).
Node.js:
A server-side JavaScript runtime that allows developers to build scalable and high-performance applications.

Based on the V8 JavaScript engine and designed to execute JavaScript code outside a web browser.

Enables the development of server-side applications, real-time applications, and APIs using JavaScript
Java:
A general-purpose, object-oriented programming language that is platform-independent.

Widely used for developing enterprise-level applications, mobile applications (Android), and web applications.

Known for its "Write Once, Run Anywhere" (WORA) philosophy, meaning Java code can run on any device with a Java Virtual Machine (JVM)
CSS (Cascading Style Sheets):
Style sheet language used to control the presentation and layout of HTML documents.

Defines how elements should be displayed, including their fonts, colors, spacing, and positioning

Allows for the separation of content and presentation, making it easier to maintain and update a website's design
React.js:
A JavaScript library for building user interfaces, particularly for single-page applications where content is dynamically updated.

Developed and maintained by Facebook.

Utilizes a component-based architecture, making it easier to manage and reuse code.

Supports a virtual DOM for efficient rendering and updating of UI components.
